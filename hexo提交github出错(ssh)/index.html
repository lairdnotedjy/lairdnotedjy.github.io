hexo d 提交github出错(ssh)

解决

  • 重新本地生成ssh-key

  • 更新,github https://github.com/settings/ssh

  • 测试连接 ssh -T git@github.com

  • hexo重新提交 hexo d

上传操作 hexo d 报错spawn git ENOENT

未添加Git环境变量引起，添加Git与git管理库的环境变量即可；
