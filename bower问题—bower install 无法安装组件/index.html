bower问题—bower install 无法安装组件

Posted on 2016-02-23 Edited on 2024-01-24 In bower

bower install 无法安装组件

原因是,组件存在git上,下载依赖git客户端

处理办法:

  • 下载 git for windows https://git-for-windows.github.io
  • 安装&配置Path
