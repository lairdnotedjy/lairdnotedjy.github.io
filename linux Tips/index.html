linux Tips

Posted on 2016-05-17 Edited on 2024-01-24

  • 查找进程
    ps -ef|grep wiki

  • 查看进程/服务，具体的安装路径 （方法一）
    ps -ef | grep java
    ps -ef | grep nginx
    ps -ef | grep tomcat

  • 查看进程/服务，具体的安装路径（方法二）
    ls -l /proc/$PID/exe ($PID为对应的pid号)

  • 停止进程
    kill -7 xxx

  • 查看那个进程占用了xxx端口
    lsof -i:xxx
