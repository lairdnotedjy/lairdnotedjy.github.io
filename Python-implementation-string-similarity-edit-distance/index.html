编辑距离

关于维基上的定义参见这里。编辑距离，又称Levenshtein距离，是指两个字串之间，由一个转成另一个所需的最少编辑操作次数。通常许可的编辑操作包括:

  • 将一个字符替换成另一个字符
  • 插入一个字符
  • 删除一个字符

例如将kitten一字转成sitting：

  • sitten （k→s）
  • sittin （e→i）
  • sitting （→g）

俄罗斯科学家Vladimir Levenshtein在1965年提出了这个概念。
